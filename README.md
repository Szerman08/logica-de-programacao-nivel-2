# Lógica de Programação — Nível 2

Exercícios avançados de lógica de programação implementados em Python, com foco em algoritmos e resolução de problemas.

## Sobre

Repositório com os exercícios do nível 2 do curso de lógica de programação. Os algoritmos são implementados em Python e abordam temas como manipulação de dados, cálculos matemáticos e lógica estruturada.

## Tecnologias

- Python 3

## Exercícios

| Arquivo | Descrição |
|---|---|
| `exec-1.py` | Troca de valores entre variáveis |
| `calculadora.py` | Calculadora com operações básicas |
| `imc.py` | Cálculo de IMC |
| `idade.py` | Cálculo e classificação de idade |
| `login.py` | Sistema simples de login |
| `exec-3` a `exec-36` | Exercícios progressivos de algoritmos |

## Como executar

```bash
python3 nome_do_arquivo.py
```
